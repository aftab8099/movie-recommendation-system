# 🎬 Movie Recommendation System

## 📌 Overview
This is a content-based movie recommendation system that suggests movies using **TF-IDF vectorization** and **cosine similarity**.

## 🚀 Features
- Recommend similar movies based on input
- Uses TF-IDF instead of Bag of Words
- Fast and efficient recommendation system

## 🧠 Tech Stack
- Python
- Pandas
- Scikit-learn

## ⚙️ How it Works
1. Data preprocessing is performed on movie dataset
2. TF-IDF vectorization is applied
3. Cosine similarity is calculated
4. Top similar movies are recommended
